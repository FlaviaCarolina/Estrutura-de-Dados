# Exercícios de Estrutura de Dados em Python 🚗👥🏢📂

Este repositório contém uma série de exercícios práticos focados na manipulação de arquivos, listas de dicionários e lógica de programação aplicada à gestão de dados.
Os desafios foram propostos pelo Prof. Roberto Porto na disciplina de Estrutura de Dados, na faculdade FAI - Centro de Ensino Superior em Gestão, Tecnologia e Educação. 

# 📋 Visão Geral

O objetivo principal é desenvolver soluções computacionais que leiam arquivos de dados (CSV, TXT) e gerem relatórios formatados, aplicando conceitos de ordenação, filtragem e cálculos financeiros.
